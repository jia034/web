Blender 簡介

Blender 是一款免費開源的 3D 創作軟體，適用於建模、動畫、模擬、渲染、合成及影片剪輯等。

全部功能

  • 3D 建模： 支援多邊形建模、雕刻、高細節建模與拓撲重構
  • 雕刻工具： 具備多種筆刷、動態拓撲與高階雕刻功能
  • 動畫與綁定： 角色骨架、Inverse Kinematics (IK)、動作捕捉
  • 物理模擬： 碰撞、布料、煙霧、液體、剛體與柔體物理
  • 渲染引擎： 支援 Cycles 與 Eevee 即時渲染
  • 視覺效果： 合成節點編輯器、多層影像處理
  • 影片剪輯： 時間軸編輯、音訊同步、轉場特效
  • 遊戲開發支援： PBR 材質、GLTF、FBX 匯出
  • Python 擴展： 可撰寫自訂腳本擴充功能

使用說明

  1. 下載與安裝： 前往 官方網站 下載最新版本並安裝
  2. 基礎操作： 熟悉 Blender 介面、視角切換與基本工具
  3. 建立物件： 使用 Add (Shift + A) 新增基本幾何形狀
  4. 編輯模式： 進入 Edit Mode (Tab) 來編輯頂點、邊與面
  5. 雕刻模式： 進行高細節雕刻並使用筆刷工具
  6. 材質與貼圖： 使用 Shader Editor 設定材質、貼圖與 PBR 效果
  7. 燈光與攝影機： 設置光源與相機視角來最佳化渲染
  8. 動畫與骨架綁定： 使用 Armature 來創建角色動畫
  9. 渲染輸出： 選擇 Cycles 或 Eevee 來進行最終渲染
  10. 影片剪輯： 在 Video Sequence Editor 內進行基本剪輯與輸出

快捷鍵

  • G： 移動物件
  • R： 旋轉物件
  • S： 縮放物件
  • Tab： 切換物件/編輯模式
  • Shift + A： 新增物件
  • X / Delete： 刪除選取物件
  • Ctrl + Z： 撤銷操作
  • Ctrl + Shift + Z： 重做操作
  • F12： 渲染圖片
  • Ctrl + B： 設定渲染區域
